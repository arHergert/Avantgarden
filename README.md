# Avantgarden

Enthält die wichtigsten Elemente für eine React App:
+ React
+ Webpack 4
+ Babel 7

Dependencies auf den aktuellen Stand von 18.12.2018

## Benutzung

Im Dev-Mode starten:
```sh
npm start
``` 
   - Hot-Reload verfügbar
   - Öffnet automatisch in Google Chrome
   
---

Im Production-Mode starten:
```sh
npm build
```
